# Portfolio Summary — subenai

## Stručný opis

subenai je webová aplikácia zameraná na digitálnu bezpečnosť. Pomáha používateľom rozpoznávať online podvody ako phishing, podvodné SMS, falošné e-shopy či manipulatívne sociálne inžinierstvo pomocou krátkych interaktívnych testov a edukačného obsahu.

## Použité technológie

- React 19
- TypeScript
- TanStack Start, TanStack Router, TanStack Query
- Vite
- Tailwind CSS v4
- Radix UI
- Supabase
- Cloudflare Pages + Pages Functions
- Stripe
- Vitest + Testing Library

## Hlavné funkcionality

- interaktívne testy na odhaľovanie online podvodov
- tematické a odvetvové test packy
- edukačné stránky s vysvetleniami konkrétnych typov útokov
- detailné vyhodnotenie odpovedí a výsledné skóre
- zdieľanie výsledkov cez share flow a generovanie vizuálu pre sociálne siete
- sponzorská / podporná sekcia s platbami cez Stripe
- cookie consent a analytics rešpektujúce voľbu používateľa
- SEO základ pre verejné landing pages vrátane sitemap a meta údajov

## Plánované funkcionality

- rozšírenie question banku o ďalšie realistické scenáre
- návrat a rozšírenie composeru pre vlastné firemné testy
- education mode pre školy a firmy so zberom výsledkov respondentov
- ďalšie odvetvové test packy a obsahové rozšírenia
- postupné vylepšovanie analytiky, onboarding flow a obsahu

## Jednovetová verzia do portfólia

subenai je full-stack edukačná webová aplikácia postavená na Reacte, TypeScripte, Supabase a Cloudflare, ktorá učí používateľov rozpoznávať online podvody cez interaktívne testy, obsahové lekcie a merateľné výsledky.
