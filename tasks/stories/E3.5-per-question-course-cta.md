# ~~E3.5 — Per-question course CTA v review karte~~ ✅

**Epic:** [E3 — Post-test answer review](../PLAN-2026-04-25-rast-a-vzdelavanie.md#epic-3)
**Effort:** `S` (≤ ½ dňa)
**Priority:** `P2` (should)
**Status:** ✅ Done (2026-04-26)

## Kontext

Review karta po teste je ideálny learning moment — používateľ
práve videl svoju chybu (alebo úspech), psychologicky je
otvorený dozvedieť sa viac. Linkujeme ho priamo na relevantný
kurz v `/kurzy/$slug` podľa kategórie otázky. CTA sa zobrazuje
aj pri správnych odpovediach (príležitosť "nauč sa viac" je
univerzálna), ale s minimalistickým copy aby sa neprepchávala.

## User story

> Ako používateľ pozerajúci si svoje odpovede po teste,
> **chcem** pri každej otázke vidieť odkaz na kurz ktorý sa
> jej týka, **aby** som sa mohol/a hneď ďalej vzdelávať
> v tej oblasti kde som pochybil/a (alebo prehĺbiť znalosť
> tam kde som zvládol/a).

## Acceptance criteria

- **AC-1**: V `<AnswerReviewCard>` pod `<AnswerFeedback>` blokom
  sa zobrazí CTA **`<Link>`** s textom **"📚 Nauč sa viac"**
  smerujúce na `/kurzy/$slug`.
- **AC-2**: Mapovanie `QuestionCategory → courseSlug` je
  centralizované v `src/lib/quiz/category-course-map.ts`:
  ```ts
  export const CATEGORY_TO_COURSE_SLUG: Record<QuestionCategory, string | null> = {
    phishing:     "email-phishing",     // E5.5
    url:          "data-hygiene",       // E5.11
    fake_vs_real: "marketplace",        // E5.7
    scenario:     "sms-smishing",       // E5.4
    honeypot:     "data-hygiene",       // E5.11
  };
  ```
- **AC-3**: Helper `getRelatedCourseForCategory(category): Course | null`
  v rovnakom súbore — vracia `Course` z registry **iba ak**
  slug je registrovaný v `COURSES` (E5.1). Ak nie → `null`.
- **AC-4**: CTA sa **nezobrazí** ak `getRelatedCourseForCategory`
  vráti `null` — graceful degradation pred dokončením všetkých
  kurzov v E5.
- **AC-5**: CTA sa zobrazuje **na všetkých kartách** — správne
  aj nesprávne odpovede, aj timeout (`optionId === null`).
- **AC-6**: Vizuál: rounded-xl border-2 border-border button-link,
  match s existujúcim "Skús znova" / share grid stylom. Hover:
  `border-primary/60`. Šírka full-width vo vnútri karty.
- **AC-7**: A11y — `<Link>` má `aria-label="Otvoriť kurz: {course.title}"`
  (kontextuálnejší než len "Nauč sa viac" pre screen readery).
- **AC-8**: TypeScript safety — `CATEGORY_TO_COURSE_SLUG` má
  exhaustívne kľúče (TS error ak sa pridá nová `QuestionCategory`
  bez updatovania mapy).

## Riziká

| Riziko | Mitigácia |
|---|---|
| Kurz pre kategóriu ešte neexistuje → broken link | `getRelatedCourseForCategory` vráti `null` → CTA sa nezobrazí; žiadny broken link nikdy nevznikne |
| Príliš veľa CTAs na obrazovke (15 kariet × 1 CTA = 15 buttonov) | Akceptovateľné — sú per-card, vizuálne diskrétne; alternatíva (jeden CTA na konci review) by stratila kontext "ktorá konkrétna otázka → ktorý kurz" |
| Mapovanie sa zmení po roku, bug v mapping table → zlý kurz | TS exhaustívnosť kľúčov + helper test "každý mapovaný slug existuje v registry" zachytí drift v CI |
| User klik na CTA stratí svoj review state (ak je expanded) | TanStack Router zachová scroll position pri back-nav; akceptovateľný UX (alternatíva: `target="_blank"` ale to fragmentuje pozornosť) |
