# E12.5 — Education author how-to guide

**Epic:** [E12 — Education mode](../PLAN-2026-04-26-custom-tests-sponsorship.md#epic-12--education-mode-autori-zbierajú-výsledky)
**Effort:** `S` (≤ ½ dňa)
**Priority:** `P2` (should)
**Status:** ✅ Done (2026-05-01)

## Kontext

Bez návodu autori nezvládnu:
- vytvoriť edu test s heslom a kolektívnym mode
- bezpečne distribuovať link
- otvoriť výsledky a interpretovať dáta
- spĺňať svoje GDPR povinnosti voči študentom (sú „kontrolóri" voči svojim študentom; my sme „spracovateľ")

## User story

> Ako učiteľ / firemný HR, **chcem** krátky čitateľný návod ako pripraviť
> test, poslať študentom a pozrieť výsledky, **aby** som nepotreboval
> kontaktovať support pri prvom použití.

## Acceptance criteria

- **AC-1**: Stránka `/skoly` (file `src/routes/skoly.tsx`) — verejná
  marketing-style + návod page.
- **AC-2**: Sekcie:
  1. **Čo to je** — 1-2 vety o edu mode use-case-och (škola, kurz, firma)
  2. **Krok 1: Vytvor test** — screenshot composer-u so šípkami, instrukcie
  3. **Krok 2: Zapni edu mode** — screenshot toggle + password setup,
     vysvetlenie že heslo si musí zaznamenať
  4. **Krok 3: Pošli link** — popis 2 linkov (public + results), formuláre
     na zdieľanie cez email šablónu pre študentov (copy-paste vzor)
  5. **Krok 4: Pozri výsledky** — screenshot dashboardu + interpretácia
     agregátov + ako exportnúť do CSV
  6. **GDPR pre teba ako autora** — autor je *kontrolór* voči študentom;
     my sme *sprostredkovateľ*; dohoda o spracovaní (DPA) je popisaná v
     [`/dpa-skoly`](#dpa) — link na sample DPA šablónu PDF
  7. **Časté otázky**:
     - Stratil som heslo — môžem ho resetovať? (Nie, nemáme reset)
     - Ako dlho sa uchovávajú odpovede? (12 mesiacov, potom anonymizácia)
     - Môžem zmazať konkrétneho študenta? (Áno, cez dashboard akciu)
     - Koľko respondentov je max? (1 000 / test_set, hard cap)
- **AC-3**: SEO:
  - `<title>Pre školy a vzdelávacie inštitúcie — subenai</title>`
  - description, OG, Twitter cards
  - Schema.org `EducationalOrganization` JSON-LD nie je vhodný (my nie sme
    organizácia); použijeme `WebPage` so service description
- **AC-4**: DPA (Data Processing Agreement) sample PDF v `/public/dpa-skoly.pdf`
  — pripravená šablóna ktorú autor podpíše (alebo my pred-podpíšeme) ak
  škola žiada formálnu dohodu. **MVP**: PDF + checkbox „Súhlasíš s DPA
  podmienkami" pri composer setup keď collects_responses=true.
- **AC-5**: Footer link „Pre školy" → `/skoly`.
- **AC-6**: Mobile-first.
- **AC-7**: A11y (semantic landmarks, alt text na screenshotoch).
