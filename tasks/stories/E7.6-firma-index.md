# E7.6 — `/test/firma` discovery page

**Epic:** [E7 — Industry test packs](../PLAN-2026-04-26-custom-tests-sponsorship.md#epic-7--industry-test-packs)
**Effort:** `S` (≤ ½ dňa)
**Priority:** `P2` (should)
**Status:** ⛔ Blocked by E7.5

## Kontext

Index stránka kde si user vyberie pack pre svoju branžu. Treba aby
bol grid filtrovateľný (industry chips), s krátkym popisom packu
a počtom otázok.

## User story

> Ako návštevník hľadajúci test pre svoju firmu, **chcem** zoznam
> všetkých industry packov s filterom podľa branže, **aby** som rýchlo
> našiel ten správny.

## Acceptance criteria

- **AC-1**: Route `src/routes/test.firma.index.tsx` na `/test/firma`.
- **AC-2**: Renderuje grid `<TestPackCard>` (analogický s `<CourseCard>`).
- **AC-3**: Filter chips podľa `industry` (kliknutie filtruje grid, multi-select).
- **AC-4**: Empty state ak filter nemá výsledky.
- **AC-5**: Hero header: „Otestuj seba alebo svoj tím" + CTA na manuálny builder.
- **AC-6**: SEO: `<title>Otestuj svoju branžu — Internet IQ Test</title>`, ItemList JSON-LD všetkých packov.
- **AC-7**: Linka na manuálny builder ako alternatíva (E8).
