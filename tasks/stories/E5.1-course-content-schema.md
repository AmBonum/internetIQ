**Epic:** [E5 — Sekcia bezplatných kurzov](../PLAN-2026-04-25-rast-a-vzdelavanie.md#epic-5)

**Priority:** `P2` (should)
**Status:** ✅ Done (2026-04-26)

## Kontext

Kurzy budú statické TS moduly (žiadny CMS — out-of-scope). Aby
sa nezabudlo na povinné polia (slug, title, sections...),
potrebujeme typovo bezpečnú schému. Schéma je single source of
truth — meniť ju je breaking change pre všetkých autorov kurzov.

## User story

> Ako vývojár pridávajúci nový kurz, **chcem** typovo bezpečnú
> štruktúru contentu, **aby** sa nezabudlo na povinné polia
> (slug, title, description, thumbnail, category, sections).

## Acceptance criteria

- **AC-1**: `src/content/courses/_schema.ts` exportuje:
  ```ts
  export type CourseCategory =
    | "sms" | "email" | "voice" | "marketplace"
    | "investicie" | "vztahy" | "data" | "obecne";

  export type Course = {
    slug: string;            // [a-z0-9-]+
    title: string;
    tagline: string;         // 1 veta pod nadpisom
    category: CourseCategory;
    difficulty: "začiatočník" | "pokročilý";
    estimatedMinutes: number;
    heroEmoji: string;
    sections: CourseSection[];
    sources?: { label: string; url: string }[];
    relatedQuestionsCategory?: "phishing" | "url" | "fake_vs_real" | "scenario";
    publishedAt: string;     // ISO 8601
    updatedAt: string;
  };

  export type CourseSection =
    | { kind: "intro";     heading: string; body: string }
    | { kind: "example";   heading: string; visual: Visual; commentary: string }
    | { kind: "checklist"; heading: string; items: { good: boolean; text: string }[] }
    | { kind: "redflags";  heading: string; flags: string[] }
    | { kind: "do_dont";   heading: string; do: string[]; dont: string[] }
    | { kind: "scenario";  heading: string; story: string; right_action: string };
  ```
- **AC-2**: `src/content/courses/index.ts` exportuje:
  ```ts
  export const COURSES: Course[] = [/* statické imports */];
  export function getCourseBySlug(slug: string): Course | null;
  export function getCoursesByCategory(c: CourseCategory): Course[];
  ```
- **AC-3**: Vytvoriť 1 placeholder kurz (`src/content/courses/_demo.ts`)
  s minimálnym validným contentom — slúži ako compile-time check
  schémy + ako reference pre autorov.
- **AC-4**: Zod schema (`courseSchema`) v rovnakom súbore — runtime
  validátor pre prípady ak by sa kurz dynamicky importoval (budúcnosť).
- **AC-5**: Slug uniqueness validovaná **v build-time** —
  `index.ts` má assertion ktorá hodí na duplicate slug:
  ```ts
  const slugs = new Set<string>();
  for (const c of COURSES) {
    if (slugs.has(c.slug)) throw new Error(`Duplicate course slug: ${c.slug}`);
    slugs.add(c.slug);
  }
  ```

## Riziká

| Riziko | Mitigácia |
|---|---|
| Schema lock-in — pridanie nového section type rozbije old courses | Variant types su rozšíriteľné, defaultná `kind` discriminácia umožňuje nové variants bez breakage |
| Zabudnem pridať kurz do `index.ts` → nezobrazí sa | Zvážiť `import.meta.glob("./*.ts")` automatickú registráciu (TanStack-Start friendly?) — alebo lint rule |
