# E7.5 — `/test/firma/$slug` route + SEO + Quiz JSON-LD

**Epic:** [E7 — Industry test packs](../PLAN-2026-04-26-custom-tests-sponsorship.md#epic-7--industry-test-packs)
**Effort:** `M` (≤ 2 dni)
**Priority:** `P1` (must)
**Status:** ⛔ Blocked by E7.1

## Kontext

Pack potrebuje verejný URL, kde sa user dostane k testu. URL je
zdielateľný cez share linky („pošli zamestnancom") + indexovaný cez
Schema.org `Quiz` JSON-LD.

## User story

> Ako majiteľ firmy zdieľajúci test, **chcem** stabilný URL ako
> `internetiq.lvtesting.eu/test/firma/eshop`, **aby** som ho mohol
> poslať tímu cez Slack/email a videl pekný preview na sociálnych
> sieťach.

## Acceptance criteria

- **AC-1**: Route `src/routes/test.firma.$slug.tsx`:
  - loader → `getPackBySlug(params.slug)` → 404 ak null
  - intro screen: pack heroEmoji + title + tagline + persona + „Spustiť pack" tlačidlo + meta (počet otázok, threshold, doba)
  - po klik na „Spustiť pack" prechádza do `TestFlow` v pack móde
- **AC-2**: `TestFlow` musí podporovať pack-mód: namiesto `getTestQuestions()` použije `pack.questionIds.map(getQuestionById).filter(Boolean)`. Žiadny shuffle (autor packu si poradie zvolil).
- **AC-3**: SEO meta:
  - `<title>{pack.title} — Internet IQ Test pre {industry}</title>`
  - description = pack.tagline
  - canonical: `https://internetiq.lvtesting.eu/test/firma/{slug}`
  - og:title, og:description, og:type=website, og:locale=sk_SK
  - twitter:card summary_large_image
- **AC-4**: Schema.org **Quiz** JSON-LD:
  ```ts
  {
    "@context": "https://schema.org",
    "@type": "Quiz",
    "name": pack.title,
    "description": pack.tagline,
    "inLanguage": "sk-SK",
    "isAccessibleForFree": true,
    "educationalLevel": "vocational",
    "about": pack.industry,
    "numberOfQuestions": pack.questionIds.length,
    "publisher": { "@type": "Organization", "name": "Internet IQ Test" }
  }
  ```
- **AC-5**: Po dokončení packu — `ResultsView` zobrazuje "Vyhovuje pre {industry}" badge ak `finalScore >= pack.passingThreshold`.
- **AC-6**: Copyright notice rovnaký ako u kurzov (zákon č. 185/2015).
- **AC-7**: Sitemap (E5.12 generator) pridáva všetky pack URL.
