## Kontext

Pre organic traffic na kurzy potrebujeme správne SEO basics:
- sitemap.xml (Google crawler hint)
- ItemList schema na `/kurzy`
- Course schema na `/kurzy/$slug`
- robots.txt nepoškodený

## User story

> Ako Google crawler, **chcem** nájsť všetky course routes
> v sitemape s aktuálnym `lastmod`, **aby** sa kurzy indexovali
> rýchlo a získali organický traffic.

## Acceptance criteria

- **AC-1**: `public/sitemap.xml` generovaný **build-time skriptom**
  `scripts/build-sitemap.ts` na základe `COURSES` registra +
  statických routes (`/`, `/test`, `/kurzy`, `/cookies`, `/privacy`).
- **AC-2**: Skript je zavolaný v `npm run build` (post-build hook
  alebo cez `build` script chain).
- **AC-3**: Každá course route v sitemap má:
  - `<loc>https://internetiq.lvtesting.eu/kurzy/{slug}</loc>`
  - `<lastmod>{course.updatedAt}</lastmod>`
  - `<changefreq>monthly</changefreq>`
  - `<priority>0.7</priority>`
- **AC-4**: `/kurzy` route má `ItemList` JSON-LD
  (Schema.org/ItemList) s každým kurzom ako `ListItem`.
- **AC-5**: Každý `/kurzy/$slug` route má `Course` JSON-LD
  (Schema.org/Course) s povinnými poľami: `name`,
  `description`, `provider`, `educationalLevel`, `timeRequired`
  (ISO 8601 duration: `PT8M`).
- **AC-6**: `public/robots.txt` aktualizovaný:
  - povoľuje `Allow: /kurzy/`
  - explicitný `Sitemap: https://internetiq.lvtesting.eu/sitemap.xml`
- **AC-7**: Per-course canonical URL nastavený v `<head>` cez
  `links: [{ rel: "canonical", href: ... }]` v Route head().

| Riziko | Mitigácia |
|---|---|
| Sitemap obsahuje neexistujúce URL po vymazaní kurzu | Skript načíta iba `COURSES` array — vymazaný kurz vypadne automaticky |

| Build hook zlyhá na CI a build padne | Skript má try/catch pre file write + clear error message |
