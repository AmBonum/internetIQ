# E11.1 — `/podpora` donate page (oneoff + monthly + Stripe Element)

**Epic:** [E11 — Sponsorship UI + invoicing](../PLAN-2026-04-26-custom-tests-sponsorship.md#epic-11--sponsorship-ui--invoicing)
**Effort:** `M` (≤ 2 dni)
**Priority:** `P1` (must)
**Status:** ⛔ Blocked by E10.3, E10.5

## Kontext

Verejne najviditeľnejšia stránka sponzorskej feature. Prepínač oneoff /
monthly, suma chips, custom input, anonymity checkbox, GDPR + odstúpenie
disclosure, Stripe Element (lazy loaded).

## User story

> Ako návštevník, ktorý si želá podporiť projekt, **chcem** transparentnú
> stránku s jasnou cenou, aký účel ide podporovať, **aby** som vedel,
> komu peniaze idú a čo dostanem na oplátku (faktúra).

## Acceptance criteria

- **AC-1**: Route `/podpora` s `<title>Podpora projektu — Internet IQ Test</title>`.
- **AC-2**: Hero: krátky text „Prečo podporiť" + cieľ použitia peňazí (hosting, content, …).
- **AC-3**: Tabs / toggle: **„Jednorazovo"** vs **„Mesačne"**.
- **AC-4**: Suma chips:
  - Jednorazovo: 5, 10, 25, 50, 100, custom (5–500)
  - Mesačne: 5, 10, 25 EUR/mes
- **AC-5**: Custom input s validáciou (`min=5`, `max=500` per AC AML hranica).
- **AC-6**: Form polia (vyžadované pre faktúru):
  - Email (povinný)
  - Meno alebo firma (povinný — pre faktúru)
  - DIČ (voliteľné, povinné iba ak je platca DPH a chce odpočet)
  - Adresa (voliteľná pri donate, povinná pri faktúre nad limit)
- **AC-7**: Checkboxy (povinné):
  - „Súhlasím so začatím poskytovania okamžite a beriem na vedomie stratu práva na odstúpenie." (per § 7 ods. 6 zákona č. 102/2014 Z.z.)
  - „Beriem na vedomie spracovanie mojich osobných údajov per [Privacy](privacy)."
- **AC-8**: Voliteľný checkbox: „Zobraziť moje meno na stránke `/sponzori`" (default OFF).
- **AC-9**: Stripe Payment Element (lazy import `@stripe/stripe-js`):
  - Card, SEPA, Apple Pay, Google Pay
  - Locale `sk`
  - On submit → POST na CF Pages Function `/api/create-checkout-session` → redirect na Stripe Checkout (alebo embedded Element + confirmPayment).
- **AC-10**: Po úspešnom payment → redirect na `/podakovanie/{stripe_session_id}` (E11.2).
- **AC-11**: Mobile-first. Bundle impact: Stripe.js loaded len pri tejto route (`React.lazy` Suspense fallback).
- **AC-12**: A11y: form labels, ARIA error messages, keyboard nav.
- **AC-13**: Žiadne dark patterns: Suma 5 EUR je rovnako prominent ako 100 EUR; default je nezvolené (user musí kliknúť).

|---|---|
