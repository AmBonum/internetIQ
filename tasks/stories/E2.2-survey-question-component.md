## Kontext

Existujúci `SurveyCard.tsx` má copy-pasted JSX pre Age range a Gender
chip-groupy (~30 riadkov × 2). Pridanie ďalších 4 otázok by zoxidovalo
súbor na ~600 riadkov. Potrebujeme abstrakciu ktorá pokryje:
- single-select (chip group, 1 odpoveď)
- multi-select (chip group, viac odpovedí)
- yes/no (binárny gate napr. `wants_courses`)
- voľný text (rare, ale pre "iný zdroj odkiaľ poznáš test")

## User story

> Ako vývojár pridávajúci nové prieskumové otázky, **chcem** jednu
> deklarat. komponentu zvládajúcu single/multi/text varianty,
> **aby** pridanie 6. otázky nepridalo 200 riadkov copy-paste JSX.

## Acceptance criteria

- **AC-1**: `src/components/quiz/SurveyQuestion.tsx` exportuje typovo
  bezpečné varianty:
  ```tsx
  <SurveyQuestion
    type="single"
    label="Pohlavie"
    options={[{ id: "m", label: "Muž" }, ...]}
    value={gender}
    onChange={setGender}
  />
  <SurveyQuestion type="multi" ... />
  <SurveyQuestion type="text" maxLength={80} ... />
  <SurveyQuestion type="yesno" ... />
  ```
- **AC-2**: Existujúce polia v `SurveyCard` (Age range, Gender)
  sú **migrované** na novú abstrakciu — žiadny vizuálny regres,
  žiadna zmena layoutu.
- **AC-3**: Klávesnicovo prístupné: Tab medzi options, Space
  toggle. ARIA: single-select má `role="radiogroup"`, multi-select
  `role="group"` s `aria-pressed` na chips.
- **AC-4**: Validation hookovateľná cez props:
  `required?: boolean`, `minSelections?: number` (multi only),
  vrátenie `error?: string` pre custom message.
- **AC-5**: Po flagnutí `error` sa zobrazí inline message pod poľom
  v destructive farbe.
- **AC-6**: Komponent nedrží vlastný state (controlled) — caller
  spravuje `value`/`onChange`.
- **AC-7**: Bundle dopad ≤ 4 KB gzipped (kontrola v build output).

## Subtasks

### Implementation
- [ ] Vytvoriť `src/components/quiz/SurveyQuestion.tsx` s
  diskriminovaným union typom pre props:
  ```ts
  type Props =
    | { type: "single"; options: Option[]; value: string; onChange: (v: string) => void; ... }
    | { type: "multi"; options: Option[]; value: string[]; onChange: (v: string[]) => void; ... }
    | { type: "yesno"; value: boolean | null; onChange: (v: boolean) => void; ... }
    | { type: "text"; value: string; onChange: (v: string) => void; maxLength?: number; ... };
  ```
- [ ] Refactor `SurveyCard.tsx` — Age + Gender groupy nahradiť
  novými `<SurveyQuestion type="single" />`. Všetky ostatné polia
  ostávajú nezmenené v tejto story.
- [ ] Pridať `Option` typ do `src/lib/quiz/survey-options.ts`
  (zdieľaný s E2.3).

### Tests
- [ ] **Unit test** per variant (`tests/components/SurveyQuestion.test.tsx`):
  - `single`: render, klik option A → onChange("A"), klik znova → onChange("") (toggle off)
  - `multi`: render, klik A → onChange(["A"]), klik B → onChange(["A","B"]), klik A → onChange(["B"])
  - `text`: typing → onChange volaný s každou zmenou, maxLength enforced
  - `yesno`: klik Áno → onChange(true), Nie → onChange(false)
- [ ] **A11y test**: každý variant prejde axe-core ruleset
  bez chýb (zero violations).
- [ ] **Validation test**: `required={true}` + empty value →
  `error` zobrazený; vyplnenie value → error zmizne.
- [ ] **Regression test**: `SurveyCard` renderuje rovnaký vizuálny
  output pre Age + Gender ako pred refaktorom (snapshot test).

### Documentation
