## Kontext

Renderovacia šablóna pre detail kurzu. Single source of truth
pre všetkých 8 kurzov (E5.4–E5.11) — každý kurz potom sype iba
content, nie layout. Senior approach: per-section komponenty
v `src/components/courses/sections/` s clear single
responsibility.

## User story

> Ako návštevník otvárajúci konkrétny kurz, **chcem** prirodzene
> scrollovateľnú stránku s úvodom → príkladmi → red flags →
> do's & don'ts → záverom, **aby** som sa to mohol/a celé
> prečítať za 5–10 min bez navigácie.

## Acceptance criteria

- **AC-1**: Route `src/routes/kurzy.$slug.tsx` načíta
  `getCourseBySlug(slug)`. Ak `null` → 404 (TanStack
  `notFoundComponent`).
- **AC-2**: Hero sekcia:
  - heroEmoji (text 7xl)
  - title (h1)
  - tagline (p)
  - meta riadok: `⏱ {estimatedMinutes} min` · `{difficulty}` · `{category}`
- **AC-3**: Pre každý `CourseSection` je vlastný renderer:
  - `IntroSection` — heading + body (markdown lite, prose)
  - `ExampleSection` — heading + `<VisualBlock>` (z QuestionCard)
    + commentary
  - `ChecklistSection` — heading + items s ✅/❌ ikonkou
  - `RedFlagsSection` — heading + flags ako červený list
  - `DoDontSection` — heading + 2-stĺpcový grid (do | dont)
  - `ScenarioSection` — heading + story + "right_action" green box
- **AC-4**: Floating Table of Contents (sticky sidebar) na desktop
  (`>=lg`):
  - jeden link per section (heading slugified)
  - aktívna sekcia highlighted (scroll-spy)
  - na mobile: collapsible "Obsah" cez Sheet
- **AC-5**: Záverečné CTA:
  - "Otestuj sa" → `/test`
  - "Súvisiace kurzy" — automaticky filtrované
    `getCoursesByCategory(course.category)` (excluding seba),
    karty v rade
- **AC-6**: SEO meta:
  - `<title>{course.title} — Bezplatný kurz · Internet IQ Test</title>`
  - description = `course.tagline`
  - og:title, og:description, og:image
  - JSON-LD `Course` schema (Google Rich Results)
- **AC-7**: Print-friendly CSS (`@media print`):
  - Hide ToC, hero emoji
  - Black text on white background
  - Sources sekcia v plnom texte (URL ako footnote)
- **AC-8**: A11y:
  - Sections majú `<section aria-labelledby="...">`
  - Sticky ToC má `aria-label="Obsah kurzu"`
  - Skip link "Preskočiť na obsah" pred ToC
- **AC-9**: Performance: žiadne section komponenty nie sú
  `React.lazy` — kurz je primárne text, eager render je správne.

## Riziká

| Riziko | Mitigácia |
|---|---|
| ToC scroll-spy laggy s 10+ sekciami | IntersectionObserver namiesto scroll-listener; threshold 0.5 |
| Section content drift — jeden kurz má unikátny block | Schema variant types — pridať nový `kind` keď reálne potrebné, nie preventívne |
| Sticky ToC zachytáva CSP issue | Plain CSS `position: sticky`, žiadne JS triky |
