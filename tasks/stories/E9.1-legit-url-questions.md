# E9.1 — +30 legitimate URL examples (banking, e-shop, eGov)

**Epic:** [E9 — Question bank +100](../PLAN-2026-04-26-custom-tests-sponsorship.md#epic-9--question-bank-100)
**Effort:** `S` (≤ ½ dňa)
**Priority:** `P1` (must)
**Status:** 🟡 Ready

## Kontext

User chce, aby quiz neučil len paranoju, ale aj rozpoznávanie *dôveryhodných*
URL. Tento batch pridá 30 honeypot-typu otázok kde URL **vyzerá podozrivo, ale
je reálne legitímne** (napr. `slovenskosporitelna-na-mojom-uctovnom-zostatku.sk`
je real domain). User sa naučí, že nie všetko s podivnou doménou je scam.

## User story

> Ako účastník testu, **chcem** vidieť aj URL ktoré sú *legitímne*, aj keď
> vyzerajú zvláštne, **aby** som sa nestal príliš paranoidný a neignoroval
> reálne dôležité oznámenia.

## Acceptance criteria

- **AC-1**: 30 nových otázok kategórie `honeypot` v `src/lib/quiz/questions.ts`.
  Distribúcia:
  - 10× banking URLs (slsp.sk, csob.sk, tatrabanka.sk subdomény)
  - 10× e-shop URLs (alza.sk, mall.sk, mojaposta.sk)
  - 10× eGov / štátne URLs (slovensko.sk, finance.gov.sk subdomény)
- **AC-2**: Každá otázka:
  - `category: "honeypot"`
  - difficulty `medium` alebo `hard` (lebo vyžaduje rozoznanie subdomény / domény)
  - `severity: "minor"` u nesprávnej odpovede (paranoja je menšie zlo než dôvera)
  - `prompt`: „Klikol by si na túto URL?" alebo podobné
  - 2 možnosti: „Áno — vyzerá legit (správna)" / „Nie — vyzerá phishingovo"
  - `explanation`: prečo je URL legit (kto ju vlastní, ako overiť)
- **AC-3**: Žiadna URL nie je reálne phishingová (žiadne typo squatting v honeypot kategórii).
- **AC-4**: Aspoň 5 otázok obsahuje subdoména/doména trick (napr. `internetbanking.tatrabanka.sk` vs `tatrabanka.internetbanking.sk`).
- **AC-5**: `getTestQuestions()` automaticky zahrnie nové otázky (žiadny refaktor).
- **AC-6**: Existujúce testy v `tests/lib/quiz/scoring.test.ts` zostávajú zelené (žiadny breaking change v scoring).

## Subtasks

### Implementation
- [ ] Research reálnych URL štruktúr SK bánk / e-shopov / eGov.
- [ ] 30 otázok do `questions.ts` s vizuálom `kind: "url"`.

### Tests
- [ ] Snapshot count: `QUESTIONS.filter(q => q.category === 'honeypot').length >= 44` (14 existujúcich + 30 nových).
- [ ] Každá honeypot URL prešla cez `new URL()` validation.

### Documentation
- [ ] Update copyright pravidiel — používame *reálne* domény SK firiem v honeypot kontexte (s pozitívnym tonom). Žiadna defamácia.

### Code review (fresh context)
- [ ] Reviewer overí, že žiadna „podozrivá" URL nie je v skutočnosti aj phishingová replica.
- [ ] Reviewer skontroluje, či explanation pre každú otázku adekvátne vysvetlí, prečo je URL legit.
