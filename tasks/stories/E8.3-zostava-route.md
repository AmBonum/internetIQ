# E8.3 — `/test/zostava/$id` shareable saved set

**Epic:** [E8 — Manual test builder](../PLAN-2026-04-26-custom-tests-sponsorship.md#epic-8--manual-test-builder)
**Effort:** `S` (≤ ½ dňa)
**Priority:** `P2` (should)
**Status:** ⛔ Blocked by E8.1, E8.2

## Kontext

Po stlačení „Zdieľať zostavu" v builderi sa test sety uloží do DB s
UUID. Tento route ho načíta a pustí ako pack.

## User story

> Ako prijímateľ zostava-linky, **chcem** otvoriť ju a hneď spustiť
> test, **aby** som videl, či splním sadu, ktorú mi niekto poslal.

## Acceptance criteria

- **AC-1**: Route `src/routes/test.zostava.$id.tsx`:
  - loader → `supabase.from('test_sets').select().eq('id', params.id).single()`
  - 404 ak nie je
- **AC-2**: Intro screen: „Niekto pre teba pripravil 12 otázok" + counter + start button.
- **AC-3**: Po štarte → `TestFlow` v pack móde (rovnaký mechanizmus ako E7.5).
- **AC-4**: Žiadny pack passing threshold (manuálny test = bez „vyhovuje" badge), len base score.
- **AC-5**: SEO: `<title>Zdieľaný test — Internet IQ Test</title>`, robots noindex (každý ID je unique, indexovať netreba).
- **AC-6**: Po dokončení testu — share link komunikuje výsledok ako bežný `attempts` riadok (s `share_id` v Supabase).
