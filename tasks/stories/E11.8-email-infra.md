**Epic:** [E11 — Sponsorship UI + invoicing](../PLAN-2026-04-26-custom-tests-sponsorship.md#epic-11--sponsorship-ui--invoicing)

## Kontext

Sponzorský flow potrebuje send-email schopnosť pre:
- Customer Portal magic link (E11.4)
- Refund notification konateľovi (E10.3 AC-10)
- Manuálne notifikácie (správca dostane info o novej donácii nad threshold)

Stripe sám pošle receipt + faktúru, ale my potrebujeme **vlastné**
emaily pre flow ktoré Stripe nerieši (magic link, ops alerty).

## User story

> Ako vývojár / DevOps, **chcem** stable transactional email pipeline
> dostupný z Cloudflare Pages Functions, **aby** sponzorské UX flow
> fungovalo end-to-end bez manuálneho zásahu.

## Acceptance criteria

- **AC-1**: Provider rozhodnutie — **Resend** (predvolené):
  - Free tier 3 000 emailov/mes (dosť pre MVP)
  - HTTP API (`POST /emails`), žiadny SMTP setup
  - DKIM + SPF auto-konfigurácia pri verifikácii domény
  - Native CF Pages Functions support (žiadny SMTP-cez-fetch hack)
  - Alternatíva: AWS SES (lacnejšie pri scale ale komplexnejšie)
- **AC-2**: DNS setup pre `lvtesting.eu`:
  - DKIM: `resend._domainkey.lvtesting.eu` TXT record
  - SPF: `v=spf1 include:resend.com -all`
  - DMARC: `_dmarc.lvtesting.eu` TXT s `p=quarantine`
  - Reverse: žiadny rDNS pre custom doménu netreba (Resend rieši)
- **AC-3**: CF Pages Function helper `functions/api/_email.ts`:
  ```ts
  export async function sendEmail(env, to: string, subject: string, html: string)
  ```
  - Používa `RESEND_API_KEY` env var
  - From: `subenai <noreply@lvtesting.eu>`
  - Reply-To: `segnities@gmail.com`
  - Bezstavový volaním Resend API; idempotency cez Resend's vlastný `idempotency_key` header.
- **AC-4**: Email šablóny v `src/content/emails/` (alebo inline pre MVP):
  - `magic-link-portal.html` — pre E11.4 Customer Portal flow
  - `refund-alert.html` — pre E10.3 konateľ notification
  - `donation-thank-you.html` — voliteľné pre /podakovanie email backup
  - Slovak language
  - Plain HTML (žiadny MJML setup pre MVP)
- **AC-5**: Env vars:
  - `RESEND_API_KEY` — secret v CF Pages env vars
  - `EMAIL_FROM` — `noreply@lvtesting.eu`
  - `EMAIL_REPLY_TO` — `segnities@gmail.com`
  - `OPS_EMAIL` — konateľov email pre ops alerty (= `segnities@gmail.com`)
- **AC-6**: Error handling — ak Resend API zlyhá:
  - Log do CF Pages Function logs
  - Pre user-facing flow (magic link) → return 500 s message „Skús neskôr"
  - Pre ops flow (refund notif) → silent fail, log only (manuálny re-trigger neskôr)
- **AC-7**: Rate limit — Resend má 10 emails/sekundu native. Naša rate-limit logika nie je potrebná pri očakávanom traffic-u <100 emailov/deň.
- **AC-8**: Anti-abuse — magic link endpoint posiela ak email valid format (regex), inak silent return 200 (anti-enumeration per E11.4 AC-3).
