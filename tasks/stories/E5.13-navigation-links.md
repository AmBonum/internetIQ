# ~~E5.13 — Navigation: Footer + Index page links na kurzy~~ ✅

**Epic:** [E5 — Sekcia bezplatných kurzov](../PLAN-2026-04-25-rast-a-vzdelavanie.md#epic-5)
**Effort:** `XS` (≤ 2h)
**Priority:** `P2` (should)
**Status:** ✅ Done (2026-04-26)

## Kontext

Kurzy musia byť **objaviteľné** zo všetkých stránok. Aktuálny
footer má len `Súkromie / Cookies / Nastavenia cookies`. Hlavná
stránka nemá žiadnu zmienku o vzdelávaní. Drobná zmena s veľkým
impactom na traffic.

## User story

> Ako návštevník, **chcem** v hlavičke alebo päte odkaz "Kurzy"
> zo všetkých stránok, **aby** som sa k nim dostal aj bez toho
> aby som dokončil test.

## Acceptance criteria

- **AC-1**: `Footer.tsx` rozšírený o link **"Kurzy"** medzi
  "Súkromie" a "Cookies" (nový riadok alebo prvý v zozname).
  Aktívny indikátor na `/kurzy*` cez TanStack Router `data-status`.
- **AC-2**: Hlavná stránka (`routes/index.tsx`) má nový blok
  "Vzdelávanie zadarmo" pod hlavným testovým CTA:
  - heading "Skús niektorý z bezplatných kurzov"
  - 3 najobľúbenejšie kurz karty (alebo random 3 z `COURSES`)
  - link "Pozri všetky →" na `/kurzy`
- **AC-3**: `ResultsView` thank-you state — ak `wants_courses=true`
  (z E2.3), zobrazí personalizovaný blok:
  - "Tu sú kurzy ktoré ťa zaujímajú:" — filtrované podľa
    `interests` array
  - inak: generic "Pozri si naše kurzy →" link
- **AC-4**: Žiadny header navbar (zatiaľ) — landing page je SPA
  scroll, nepotrebuje sticky nav. Footer + main CTA stačia.

## Závislosti

- **Blokuje:** E5.2 (`/kurzy` musí existovať pred linkom)
- **Soft** závislosť na E2.3 (`wants_courses`/`interests` polia)
  — ale link funguje aj bez nich (generic fallback)

## Riziká

| Riziko | Mitigácia |
|---|---|
| Vzdelávanie sekcia odbúrava test CTA | A/B test: meranie test-completion rate pred + po |
| Random 3 kariet skreslí "popularita" → traffic na náhodný kurz | V budúcnosti nahradiť skutočným ranking podľa course completion analytics |
