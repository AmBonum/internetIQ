# ~~E5.2 — `/kurzy` — index route~~ ✅

**Epic:** [E5 — Sekcia bezplatných kurzov](../PLAN-2026-04-25-rast-a-vzdelavanie.md#epic-5)

**Priority:** `P2` (should)
**Status:** ✅ Done (2026-04-26)

## Kontext

Centrálna route ktorá zobrazí všetky kurzy z `COURSES` registra
v gridze, s filtrovaním podľa kategórie. Najdôležitejšia stránka
pre SEO (`ItemList` schema, organic traffic).

## User story

> Ako návštevník, **chcem** na `/kurzy` vidieť prehľadný zoznam
> všetkých kurzov rozdelený podľa kategórie, **aby** som si vybral
> ten, ktorý ma chráni proti najreálnejšej hrozbe.

## Acceptance criteria

- **AC-1**: Route `src/routes/kurzy.tsx` (TanStack file-route).
  URL: `/kurzy`.
- **AC-2**: Hero sekcia: H1 "Bezplatné kurzy: nedaj sa oklamať",
  1-vetný value-prop, CTA "Začni testom" → `/test`.
- **AC-3**: Filter chips: "Všetko" + 8 kategórií (sms, email,
  voice, marketplace, investicie, vztahy, data, obecne).
  Default `Všetko`. State držaný v search-params (`?cat=sms`)
  pre zdieľateľné URL filtre.
- **AC-4**: Course karta zobrazuje:
  - heroEmoji (text 5xl)
  - title (h3)
  - tagline (p, muted)
  - badge: difficulty
  - badge: kategória
  - meta: estimatedMinutes ("⏱ 8 min")
- **AC-5**: Klik na kartu → `<Link to="/kurzy/$slug" params={{
  slug: course.slug }}>`.
- **AC-6**: SEO meta:
  - `<title>Bezplatné kurzy obrany pred online podvodmi · Internet IQ Test</title>`
  - description (1 veta)
  - og:image (statický `og-courses.png` v public/)
  - JSON-LD `ItemList` s každým kurzom ako `ListItem`
    (linkuje na `Course` schema na detail page)
- **AC-7**: Empty state: ak filter nevracia žiaden kurz, zobrazí
  "Žiadny kurz v tejto kategórii — pozri všetky" tlačidlo
  reset-filtra.
- **AC-8**: Layout responzívny: 1 stĺpec (mobile), 2 stĺpce
  (sm), 3 stĺpce (lg).
- **AC-9**: Footer reused (Footer komponent).
